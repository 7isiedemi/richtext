浏览器端

▶ 操作说明

📹 导入视频

拖拽视频到页面中央，或点击「选择文件」。支持 MP4 / WebM / MOV。

✄ 剪切删除

时间轴上鼠标拖拽选中片段 → 点击红色「剪切选区」。列表中点 ✕ 撤销。

⏰ 变速

全局：右侧面板拖滑块。区间：时间轴拖选 → 绿色「变速选区」→ 选速度。

💬 字幕

时间轴拖选 → 「添加字幕」→ 输入文本。样式（字号/位置/颜色/底色）为全局设置，修改立即生效。

🎤 配音

选择语音角色 → 调语速音调 → 「预览配音」。推荐用 Edge 浏览器获取更多高品质语音。

🌈 渐隐过渡

默认开启，开头/结尾各 0.5 秒黑屏渐变。取消勾选即关闭。

🖶 输出尺寸

默认 1400×760（上下补黑边），可切换 1080p / 720p / 480p。

📥 导出

点击「导出 MP4 视频」→ 等待录制转码 → 自动下载。首次需联网加载 FFmpeg。

⌨ 快捷键

空格播放/暂停
← →快退/快进 5秒
Esc清除选区
滚轮缩放时间轴
